RF2 Authoring Services.

This service allows new concepts and value sets to be created within the context of the CIMI namespace (1000160). The CIMI module identifier is 11000160102. Value sets can then be dumped in RF2 format for distribution.

Workflow

RF2 Authoring Services.

Function Description Signature Parameters
New ChangeSet Create a new changeset. If successful, a redirect will be issued to the new changeset record record. POST /changeset
csnameThe unique change set name, which can be used in place of the UUID in other calls. Default: the changeset UUID
ownerThe change set owner. Optional
descriptionA description of the purpose or intent of the change set. Optional
formatReturn format for the completed changeset. Choices: 'html', 'json', 'xml'. Default: html

formatReturn format for the completed changeset. Choices: 'html', 'json', 'xml'. Default: html
Read ChangeSet Details. Note that there currently isn't an HTML representation for details so "xml" or "json" will have to be explicitly stated. Retrieve the records in an existing changeset GET /changeset/{changeset uuid or name}/details
formatReturn format for the completed changeset. Choices: 'json', 'xml'. Default: html
